## App Client

### Installation

To install dependencies, run:
```bash
npm install
```

### Configure Reown

Copy the example environment file:
```bash
cp .env.example .env
```
Then, fill in the `VITE_PROJECT_ID` in the `.env` file. Get projectId from https://cloud.reown.com

### Start the Client

Run the following command to start the client:
```bash
npm run dev
```
